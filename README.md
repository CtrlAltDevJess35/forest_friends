Forest Friends 🌲🐾
A mobile game developed with Flutter & Flame.

📖 About the project
‘Forest Friends’ is a project developed as part of my studies (B.Sc. Game Design & Production). It combines playful design with technical efficiency. The goal was to create a high-performance, cross-platform application that demonstrates how modern frameworks such as Flutter can be used for game development.

🛠 Tech Stack
Framework: Flutter (for a consistent UI/UX)

Engine: Flame Engine (for 2D game logic)

Language: Dart

Architecture: Modular design for easy extensibility and clean code.

✨ Key Features
Cross-platform: Runs natively on Android and webbrowser from a single codebase.

Clean architecture: Clear separation between game logic, UI components and data management.
